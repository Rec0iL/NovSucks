# SSX303

Novritsch SSX303 — a carbine conversion of the [SSX23](../ssx23/) (MK23-style green gas NBB pistol). The SSX303 is not a distinct platform; it's the same SSX23/MK23 internals housed in a carbine body kit.

## Known quirks / notes

- Like any gas-powered replica (GBB or NBB), consecutive shots lose consistency as gas pressure/temperature drops over a mag dump — this isn't specific to the NBB system, it's inherent to gas power in general.
- Recommended fix: run it on an **HPA adapter** instead of green gas/CO2. HPA removes the gas-consistency problem entirely, and precision improves substantially once converted. A shotgun-shell-style HPA adapter is a clean, non-destructive way to do this: the adapter itself supplies the HPA air, while the shotgun shells feed BBs (they carry no gas/air) — see [links.md](links.md).
- The shotgun-shell format's main advantage over other HPA adapters that borrow a different platform's magazine (M4, MP5, ARP9, etc.) is that it doesn't increase the grip/mag height — the shell sits in the original MK23-profile mag, so the replica's silhouette stays unchanged.
- On HPA, tested up to 2 Joules with correspondingly better range; it likely has headroom up to ~3J, though most fields cap DMR-style weapons like this at 2.5J — tune within your field's legal limit.
