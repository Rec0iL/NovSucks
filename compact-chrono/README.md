# Compact Chrono (E308)

Novritsch Compact Chrono, model **E308** — pocket-sized chronograph. 73x51x35mm, measures FPS/MPS/Joule, USB-C rechargeable, reinforced aluminum plates, backlit display, configurable ammo parameters, auto sleep.

Source: [eu.novritsch.com/de/product/compact-chrono](https://eu.novritsch.com/de/product/compact-chrono/)

## Known quirks / notes

- Just a rebrand of the HT-X300X without wifi.
