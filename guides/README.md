# Guides

Cross-cutting how-tos that aren't tied to a single product. Add one Markdown file per guide, e.g.:

- `identifying-the-oem-source.md` — how to figure out which AliExpress/OEM factory a given part or product actually comes from
- `tools-and-equipment.md` — calipers, printers, materials, and other tools useful for this repo's work
- `3d-printing-tips.md` — general settings/material advice for airsoft parts (strength vs. weight tradeoffs, etc.)

See [CONTRIBUTING.md](../CONTRIBUTING.md) for how product-specific content differs from what belongs here.
