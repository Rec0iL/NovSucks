# SSX23

Novritsch SSX23 — MK23-style green gas NBB (non-blowback) pistol. Also sold in a carbine body kit as the [SSX303](../ssx303/), which shares the same internals.

## Known quirks / notes

- Like any gas-powered replica, consecutive shots lose consistency as gas pressure/temperature drops over a mag dump. The same HPA adapters as the [SSX303](../ssx303/) fit here (see [links.md](links.md)), but since the SSX23 is typically run as a sidearm/backup at shorter range, the precision loss matters much less in practice — converting it isn't that sensible unless you specifically want the upgrade.
